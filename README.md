# Rocnikovy-projekt
Projekz o výtvoru hry inspirovaná indie hrou Undertale.

# Cíl
Cíl tohoto projektu je vytvořit hru podobné hře undertale. Udělat pohlcující boj a zakombinovat do toho zajímavý příběh.

# Program a programovací jazyk
Používám program Gamemaker studio 2, který používá svůj vlastní jazyk.
